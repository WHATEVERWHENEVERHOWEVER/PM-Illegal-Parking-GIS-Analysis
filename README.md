# PM-Illegal-Parking-GIS-Analysis
# 🛴 영등포구 개인형 이동장치(PM) 불법 주차 공간 데이터 시각화 분석

## 📌 프로젝트 개요
강화된 단속에도 불구하고 전동 킥보드(PM) 불법 주차가 줄지 않는 구조적 원인을 찾기 위해 진행한 데이터 탐사 프로젝트입니다. 서울시 공공데이터와 행정구역 로우 데이터를 수집 및 정제하여, 주차 공간 인프라와 실제 견인 다발 구역의 괴리를 지리 정보 시스템(GIS)으로 시각화했습니다.
* **수상 내역:** SBS 데이터사이언스와저널리즘아카데미 4기 우수상 수상작

## 🛠 사용 기술 (Tech Stack)
* **Language:** Python
* **Data Analysis & GIS:** Pandas, GeoPandas, Folium
* **Data Source:** 서울시 공공데이터, 정보공개청구 데이터

## 💡 코드의 핵심 역할 및 수행 업무
1. **공공데이터 전처리 및 병합 (`영등포구 주차구역 데이터 분석.ipynb`)**
   * 영등포구 견인 누적 데이터와 주차 구역 현황 데이터를 수집하여 결측치를 처리하고 분석 가능한 형태로 정제.
   * 대중교통 인프라(지하철역, 버스정류장) 좌표 데이터를 추출하여 PM 주차 패턴과의 상관관계 분석 기반 마련.
2. **공간 데이터 매핑 및 시각화 (`GIS.SEOULALL.NEW.ipynb`)**
   * Folium과 GeoPandas를 활용해 영등포구 행정구역 폴리곤 위에 불법 견인 좌표를 시점별(24년~25년)로 매핑.
   * **데이터 무결성 검증:** 행정구역 폴리곤 경계와 실제 수치가 엇갈리거나 좌표가 오매칭되는 시각화 오류를 방지하기 위해, 지역명과 좌표 데이터를 철저히 교차 검증하여 저널리즘적 팩트의 정확도 확보.

## 📊 시사점
영등포구 내 견인 누적 건수는 6,648건에 달하나 실제 주차구역은 0곳임을 데이터로 입증. 단속만 강화하고 인프라는 공급하지 않는 행정의 구조적 맹점을 시각적으로 증명했습니다.
